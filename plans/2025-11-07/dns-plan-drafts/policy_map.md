# Policy Map

- **ICANN Registrar Accreditation Agreement (2013)** – DNS change authorization and contact validation requirements.
- **CA/B Forum Baseline Requirements v2.0.2** – DNS challenges for certificate issuance.
- **NIST SP 800-63B** – Email security recommendations supporting MFA and phishing resistance.
- **Kenya Data Protection Regulations (for property status list)** – Requirement to host data within jurisdiction when applicable.
- **Canadian Anti-Spam Legislation (CASL)** – SPF/DKIM/DMARC alignment for communications.
- **W3C DID Specification** – DID documents served via `_did` subdomains.

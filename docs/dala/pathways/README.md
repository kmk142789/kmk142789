# Digital Autonomous Legal Atlas (DALA) Pathways

This directory captures procedural playbooks for supportive or neutral jurisdictions cataloged in `data/dala/jurisdictions.json`. Each pathway file details:

1. Available corporate, nonprofit, and DAO-oriented wrappers for digital-native projects.
2. Required documents, filings, fees, and expected timelines.
3. Operational notes to maintain compliance across token, treasury, and governance activities.
